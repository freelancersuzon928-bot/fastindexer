🔑 IndexNow Key কীভাবে পাবেন?

1
indexnow.org তে যান এবং একটি key তৈরি করুন। অথবা যেকোনো random string (32+ character) key হিসেবে ব্যবহার করতে পারেন।
2
আপনার key টি দিয়ে একটি .txt file তৈরি করুন। File এর নাম হবে your-key.txt এবং ভেতরে শুধু key টি লিখুন।
3
ওই .txt file টি আপনার site এর root এ রাখুন। যেমন: https://yoursite.com/your-key.txt
4
নিচে আপনার key ও site host দিয়ে Save করুন। তারপর URL submit করুন।
