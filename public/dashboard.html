🔑 IndexNow Key কীভাবে পাবেন? (একবারই করতে হবে)

1
indexnow.org এ যান → Key generate করুন।
2
সেই key দিয়ে একটা file বানান: your-key.txt — ভেতরে শুধু key লিখুন।
3
File টি আপনার site এর root এ রাখুন: https://yoursite.com/your-key.txt
4
নিচে key ও host একবার save করুন। এরপর শুধু URL দিলেই submit হবে!
