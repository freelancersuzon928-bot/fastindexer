🔑 Website Verify করুন (IndexNow Key Setup)
⚠️ URL Submit করার আগে এটি করা আবশ্যক! Website verify না করলে কোনো URL submit হবে না। নিচের গাইড দেখে মাত্র ১-২ মিনিটে সহজেই করে ফেলুন।
ℹ️ কেন করতে হবে? IndexNow হলো Microsoft-এর approved একটি protocol। এটি দিয়ে search engine-কে জানানো হয় আপনার সাইটে নতুন content এসেছে। আপনার site-এ একটি key রাখলেই প্রমাণ হয় যে সাইটটি আপনার — এরপর আর কোনো ঝামেলা ছাড়াই URL submit করতে পারবেন।
✅ বর্তমান Site: -  |  Key: -
উদাহরণ: https://yoursite.com অথবা https://yoursite.blogspot.com
নিচের গাইড দেখে আপনার site type অনুযায়ী key সংগ্রহ করুন এবং এখানে paste করুন।
📖 আপনার Site Type বেছে নিন — Step-by-Step Guide:
🟦 WordPress
🟠 Blogger
🌐 অন্য Site
1
আপনার WordPress Dashboard-এ login করুন। বাম মেনু থেকে Plugins → Add New এ যান।
2
Search box-এ IndexNow লিখুন। IndexNow (by Microsoft Bing) plugin খুঁজে Install Now চাপুন, তারপর Activate করুন।
3
Activate হলে বাম মেনুতে Settings → IndexNow এ যান। সেখানে একটি API Key দেখতে পাবেন — সেটি copy করুন।
4
উপরে Website URL বক্সে আপনার সাইটের লিঙ্ক এবং IndexNow Key বক্সে কপি করা key-টি paste করে 💾 Save করুন বাটন চাপুন।
✅ হয়ে গেল! এখন Dashboard থেকে সরাসরি URL Submit করতে পারবেন।
1
প্রথমে উপরে IndexNow Key বক্সে যেকোনো একটি key লিখুন — যেমন: fastindex123abc (letters ও numbers মিলিয়ে)।
2
নিচের Code-টি copy করুন (আপনার key দিয়ে auto-generate হবে):
<meta name="indexnow-key" content="your_key_here"/>
3
Blogger Dashboard → Theme → Edit HTML এ যান। HTML-এর <head> লেখার ঠিক নিচে copy করা code-টি paste করুন। তারপর Save করুন।
4
এবার উপরে Website URL ও IndexNow Key সঠিকভাবে দিয়ে 💾 Save করুন চাপুন।
✅ সফলভাবে সেটআপ সম্পন্ন! এখন URL Submit করতে পারবেন।
ℹ️ Wix, Squarespace, Webflow, Custom HTML Site — সব ক্ষেত্রেই একই meta tag পদ্ধতি কাজ করবে।
1
উপরে IndexNow Key বক্সে একটি key লিখুন (যেকোনো letters+numbers, যেমন: mykey2025abc)।
2
নিচের code-টি copy করুন:
<meta name="indexnow-key" content="your_key_here"/>

এই code-টি আপনার site-এর <head> section-এ যোগ করুন:
• Wix: Settings → Custom Code → Head
• Webflow: Project Settings → Custom Code → Head Code
• Custom HTML: সরাসরি <head> tag-এর ভেতরে paste করুন।
3
Site publish করুন। তারপর উপরে Website URL ও Key দিয়ে 💾 Save করুন চাপুন।
✅ Done! এখন যেকোনো URL submit করতে পারবেন।

❓ সাধারণ প্রশ্ন (FAQ)
১. Website verify করা কি বাধ্যতামূলক?
হ্যাঁ, সম্পূর্ণ বাধ্যতামূলক। Search engine policy অনুযায়ী, আপনি যে site-এর URL submit করছেন তা আপনার কিনা তা IndexNow key দিয়ে প্রমাণ করতে হয়। Verify ছাড়া কোনো URL submit হবে না।
২. Free plan-এ কতটি URL index করা যাবে?
Free plan-এ সর্বোচ্চ ৫টি URL one-time index করা যাবে। Website verify করার পরই ৫টি submit করতে পারবেন। এর বেশি করতে চাইলে Paid Plan নিতে হবে।
৩. একবার key save করলে কি বারবার করতে হবে?
একেবারেই না! একবার save করলেই যথেষ্ট। পরবর্তীতে আপনি সেই domain-এর যেকোনো URL সরাসরি submit করতে পারবেন।
