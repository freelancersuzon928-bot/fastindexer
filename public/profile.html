🔑 IndexNow Key সেটআপ করুন
ℹ️ কেন করতে হবে? IndexNow হলো একটি standard protocol যা দিয়ে আপনার site থেকে search engine-কে জানানো হয় যে নতুন content এসেছে। একবার key verify করলেই বারবার করতে হবে না — এরপর যত URL ইচ্ছা submit করতে পারবেন।
✅ বর্তমান Site: -  |  Key: -
উদাহরণ: https://yoursite.com অথবা https://yoursite.blogspot.com
নিচের guide দেখে আপনার site type অনুযায়ী key সংগ্রহ করুন এবং এখানে paste করুন।
📖 আপনার site type বেছে নিন:
🟦 WordPress
🟠 Blogger
🌐 অন্য Site
1
WordPress-এ login করুন। বাম মেনু থেকে Plugins → Add New এ যান।
2
Search box-এ IndexNow লিখুন। IndexNow (by Microsoft Bing) plugin খুঁজে Install Now চাপুন, তারপর Activate করুন।
3
Activate হলে বাম মেনুতে Settings → IndexNow এ যান। সেখানে একটি API Key দেখতে পাবেন — সেটি copy করুন।
4
উপরে আপনার Website URL এ আপনার site address এবং IndexNow Key এ copy করা key paste করুন। তারপর Save করুন button চাপুন।
✅ হয়ে গেল! এখন থেকে URL Submit করতে পারবেন।
1
প্রথমে একটি IndexNow key তৈরি করুন। যেকোনো সহজ word+number মিলিয়ে লিখুন।
উদাহরণ: myblog2024key বা fastindex123abc

এই key-টি মনে রাখুন বা copy করে রাখুন।
2
Blogger Dashboard-এ যান → Theme → Edit HTML button এ click করুন।
3
HTML code এর একদম শুরুতে <head> লেখাটির ঠিক নিচের লাইনে নিচের code টি paste করুন:

<meta name="indexnow-key" content="আপনার_key_এখানে_লিখুন"/>
আপনার_key_এখানে_লিখুন এর জায়গায় step 1-এ তৈরি করা key লিখুন।
4
উপরে Save বা disk icon চাপুন — theme save হয়ে যাবে।
5
এখন উপরের form-এ Website URL (আপনার Blogger site address) এবং IndexNow Key (step 1-এর key) দিয়ে Save করুন চাপুন।
✅ হয়ে গেল! এখন URL Submit করতে পারবেন।
ℹ️ Wix, Squarespace, Webflow, Custom Site — সব ক্ষেত্রেই একই পদ্ধতি।
1
একটি key তৈরি করুন — যেকোনো letters ও numbers মিলিয়ে লিখুন।
উদাহরণ: mysite2024abc
2
আপনার site-এর HTML head section এ এই meta tag টি যোগ করুন:

<meta name="indexnow-key" content="আপনার_key"/>
Wix হলে: Settings → Custom Code → Head এ paste করুন।
Webflow হলে: Project Settings → Custom Code → Head Code।
অন্য platform হলে সেখানকার SEO বা Custom Code section দেখুন।
3
Site publish/save করুন। তারপর উপরের form-এ Website URL ও IndexNow Key দিয়ে Save করুন চাপুন।
✅ হয়ে গেল! এখন URL Submit করতে পারবেন।
