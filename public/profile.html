🔑 IndexNow Key সেটআপ করুন
ℹ️ কেন করতে হবে? IndexNow হলো একটি standard protocol যা দিয়ে আপনার site থেকে search engine-কে জানানো হয় যে নতুন content এসেছে। আপনার সাইট ভেরিফাই করার জন্য শুধু প্রোফাইলে আপনার সাইটের সঠিক API Key এবং URL দিয়ে সেভ করতে হবে। এরপর কোনো রুট অ্যাক্সেস বা ফাইল আপলোডের ঝামেলা ছাড়াই আপনি আনলিমিটেড ইনডেক্স করতে পারবেন।
✅ বর্তমান Site: -  |  Key: -
উদাহরণ: https://yoursite.com অথবা https://yoursite.blogspot.com
নিচের guide দেখে আপনার site type অনুযায়ী key সংগ্রহ করুন এবং এখানে paste করুন।
📖 আপনার site type বেছে নিন:
🟦 WordPress
🟠 Blogger
🌐 অন্য Site
1
WordPress-এ login করুন। বাম মেনু থেকে Plugins → Add New এ যান।
2
Search box-এ IndexNow লিখুন। IndexNow (by Microsoft Bing) plugin খুঁজে Install Now চাপুন, তারপর Activate করুন।
3
Activate হলে বাম মেনুতে Settings → IndexNow এ যান। সেখানে একটি API Key দেখতে পাবেন — সেটি copy করুন।
4
উপরে আপনার Website URL বক্সে আপনার সাইটের লিঙ্ক এবং IndexNow Key বক্সে কপি করা কী-টি পেস্ট করে Save করুন বাটন চাপুন।
✅ হয়ে গেল! এবার কোনো রুট সিস্টেম ছাড়াই সরাসরি URL Submit করতে পারবেন।
1
প্রথমে একটি IndexNow key তৈরি করুন বা যেকোনো random letters ও numbers মিলিয়ে ওপরে ইনপুট দিন (যেমন: fastindex123abc)।
2
Blogger Dashboard-এ যান → Theme → Edit HTML বাটনে ক্লিক করুন।
3
HTML code এর একদম শুরুতে <head> লেখাটির ঠিক নিচের লাইনে নিচের জেনারেট হওয়া কোডটি পেস্ট করুন:
<meta name="indexnow-key" content="your_key_here"/>
4
Blogger থিমটি Save করুন। এবার আমাদের প্যানেলের ওপরে Website URL এবং IndexNow Key সঠিক তথ্য দিয়ে Save করুন বাটন চাপুন।
✅ সফলভাবে সেটআপ সম্পন্ন! এখন আপনি ফ্রিতে ইনডেক্সিং রিকোয়েস্ট পাঠাতে পারবেন।
ℹ️ Wix, Squarespace, Webflow, Custom Site — সব ক্ষেত্রেই একই মেটা ট্যাগ পদ্ধতি প্রযোজ্য।
1
যেকোনো একটি Key তৈরি করে ওপরের বক্সে বসান।
2
আপনার সাইটের HTML Head Section-এ গিয়ে নিচের মেটা ট্যাগ কোডটি যোগ করুন:
<meta name="indexnow-key" content="your_key_here"/>

* Wix হলে: Settings → Custom Code → Head এ বসান।
* Webflow হলে: Project Settings → Custom Code → Head Code।
3
সাইট পাবলিশ করুন এবং ওপরে Website URL ও Key দিয়ে Save করুন চাপুন।

❓ Frequently Asked Questions (FAQ)
১. ওয়েবসাইট ভেরিফিকেশন করা কেন বাধ্যতামূলক?
সার্চ ইঞ্জিন পলিসি অনুযায়ী, আপনি যে ওয়েবসাইটের লিংক সাবমিট করছেন তা সত্যিই আপনার কি না তা নিশ্চিত করতেই IndexNow Key ভেরিফাই করতে হয়। এটি স্প্যামিং রোধ করে।
২. ফ্রি ট্রায়ালে কি IndexNow Key ছাড়া সাবমিট করা যাবে?
না, ফ্রি ট্রায়াল বা যেকোনো প্ল্যানে সাবমিট করার আগে প্রোফাইল থেকে অবশ্যই আপনার "IndexNow Key" এবং "Site Host" সেভ করে নিতে হবে। অন্যথায় সিস্টেম রিকোয়েস্ট রিজেক্ট করবে।
৩. একবার কী সেভ করার পর কি বারবার করতে হবে?
একেবারেই না! একবার আপনার ডোমেইনের সাথে কী ভেরিফাই করে নিলে, পরবর্তীতে আপনি ওই ডোমেইনের যেকোনো ইউআরএল সরাসরি এক ক্লিকে সাবমিট করতে পারবেন।
