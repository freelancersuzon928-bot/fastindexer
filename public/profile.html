📖 IndexNow Key কিভাবে পাবেন ও Setup করবেন?
1

IndexNow Key তৈরি করুন

নিচের যেকোনো site এ গিয়ে আপনার IndexNow API Key জেনারেট করুন:

• Bing IndexNow — সবচেয়ে সহজ
• Google Search Console
• Yandex IndexNow

অথবা যেকোনো 32-128 character এর random hex string তৈরি করুন।

2

Key File আপনার Website এ বসান

আপনার website এর root folder এ একটি text file রাখতে হবে।
File নাম হবে: {your-key}.txt

উদাহরণ — যদি আপনার key হয় abc123xyz তাহলে:

https://yourdomain.com/abc123xyz.txt

File এর ভেতরে শুধু আপনার key লিখুন:

abc123xyz
3

WordPress ব্যবহারকারীদের জন্য

WordPress site হলে সবচেয়ে সহজ উপায়:

• Rank Math SEO Plugin ইন্সটল করুন
• Settings → General → IndexNow এ যান
• Auto-generate key অপশন চালু করুন
• Generated key টি এখানে copy করে paste করুন

4

উপরের form এ Key ও Host দিন

• IndexNow Key: আপনার generated key
• Site Host: শুধু domain name (https:// ছাড়া)

Key: a1b2c3d4e5f6789012abcdef
Host: yourdomain.com

Save করার পর Dashboard থেকে URL submit করতে পারবেন।

⚠️ গুরুত্বপূর্ণ: Key file আপনার website এ না বসালে IndexNow কাজ করবে না। File টি publicly accessible হতে হবে।
